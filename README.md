# CommonGrounds

CommonGrounds is a modern, all-in-one dashboard designed to help students organize their academic life. It provides a centralized platform to manage tasks, track class schedules, monitor learning progress, and stay focused.

## ‚ú® Features

- **Interactive Dashboard:** Get a quick overview of your upcoming tasks, today's classes, and module progress.
- **Learning Hub:** A dedicated space to manage all your academic modules and tasks. You can filter tasks, add new ones, and view your progress.
- **Calendar View:** Keep track of your schedule, deadlines, and important events.
- **Focus Mode:** A special mode to help you concentrate on your studies without distractions.
- **Responsive Design:** The application is designed to work seamlessly across various screen sizes, from desktops to mobile phones.
- **AI Assistant:** An integrated AI chat feature to help you with your questions.
